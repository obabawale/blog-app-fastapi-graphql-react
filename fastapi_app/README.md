# FastAPI Blog application

### 1. Clone the Repository
```bash
git clone git@github.com:obabawale/blog-app-fastapi-graphql-react.git
``` 

### 2. Go into the fastapi project
```bash
cd fastapi_app
```
### 3. If you have not created the virtual environment, create it.
```bash
python -m venv env
```

### 4. Activate the virtual environment
```bash
source env/bin/activate
```

### 5. Install requirements
```bash
pip install -r requirements.txt
```

### 6. Start the app
```bash
uvicorn app:app --reload
```
